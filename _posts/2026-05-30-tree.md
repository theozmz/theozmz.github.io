vector<int> preorder(TreeNode* root) {
	vector<int> ans;

		return ans;
	}
	auto dfs = [&](this auto&& dfs, TreeNode* root) {

## 迭代

迭代相对复杂一些，但是也存在一个相对固定的模板，变化的是**记录答案的位置**。需要特别注意的是**后序遍历**，由于遍历完右节点需要知道根节点是什么，所以需要用`pre`记录根节点。

### 前序遍历

```c++
vector<int> preorder(TreeNode* root) {
	vector<int> ans;
	if (root == nullptr) {
		return ans;
	}
	stack<TreeNode*> stk;
	while (root != nullptr || !stk.empty()) {
		while (root != nullptr) {
			ans.push_back(root->val); // here
			stk.push(root);
			root = root->left;
		}
		root = stk.top();
		stk.pop();
		root = root->right;
	}
	return ans;
}
```

### 后序遍历

这里需要额外记录上一个节点，并且判断当前根节点是否需要重复入栈。

```c++
vector<int> postorder(TreeNode* root) {
	vector<int> ans;
	if (root == nullptr) {
		return ans;
	}
	stack<TreeNode*> stk;
	TreeNode* pre = nullptr; // here
	while (root != nullptr || !stk.empty()) {
		while (root != nullptr) {
			stk.push(root);
			root = root->left;
		}
		root = stk.top();
		stk.pop();
		if (root->right == nullptr || root->right == pre) { // here
			ans.push_back(root->val); // here
			pre = root;
			root = nullptr;
		}
		else { // here
			stk.push(root);
			root = root->right;
		}
	}
	return ans;
}
```
